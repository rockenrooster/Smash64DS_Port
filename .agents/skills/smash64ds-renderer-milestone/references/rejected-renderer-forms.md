new design changes the measured cost model:

- command/opcode VM that still calls generic VTX/TRI helpers;
- generic fixed packet cache;
- 121 small `glCallList` submissions;
- isolated no-Z or raw triangle kernel presented as a complete owner;
- CPU joint preorder as the main M2 cut;
- split/hierarchy-only matrix cut;
- per-joint GX hierarchy packet;
- DS hardware one-light substitution for source-exact fighter lighting;
- whole-owner FIFO packet copied/prepared every frame;
- per-vertex redundant state suppression;
- final composed frame replay;
- static water prewarm that exceeds exact VRAM capacity.
