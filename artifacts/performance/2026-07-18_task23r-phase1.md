## Decision

but a cache and its record/code layout change the DS working set. The Task 23R
policy therefore requires retail falsification before promotion. No additional
retail run is available, and the candidate window also did not exercise the
required sampled recompute path under profile 2. The candidate mechanism,
selector, globals, BSS, and code were removed. Retained production footprint is
zero; only the compile-gated residual census and its checker/harness export
remain. The reverted tree passes the full Boundary verifier in 165.3 seconds;
